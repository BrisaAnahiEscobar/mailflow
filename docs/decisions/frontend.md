## Login en dos pasos

Fecha

2026-07

Estado

Aceptada

### Decisión

El inicio de sesión y la autorización de Gmail se presentan como dos pantallas independientes.

### Motivo

Permite explicar claramente qué permisos solicita la aplicación y mejora la experiencia para usuarios con necesidades de
accesibilidad.

Además, en Web Google requiere una interacción explícita del usuario para solicitar determinados scopes.

### Alternativas

Realizar el flujo completo automáticamente.

### Resultado

Descartado por menor transparencia.
